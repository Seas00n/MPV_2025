# MPV_2025
目前只支持Ubuntu20.04 ROS 

## 环境配置

```
cd /path/to/MPV_2025/
catkin build 
```

缺少的lib安装即可

## mpv_control控制
需要安装lcm
```
https://lcm-proj.github.io/lcm/content/install-instructions.html
```
